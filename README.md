# Login Form

Login Form Built Using HTML, CSS and JavaScript.

![Login Form Screenshot](./Screenshot%202024-07-03%20085631.png)

## Overview

This repository contains a simple, responsive login form built with plain HTML, CSS, and a small amount of JavaScript. It's intended as a lightweight UI example or starter for learning front-end basics.

Languages used
- CSS: 62.7%
- HTML: 37.3%

## Features
- Clean, modern login UI
- Responsive layout that works on desktop and mobile
- Client-side form validation (basic)
- Easy to customize styles and markup

## Files
- `index.html` — the login page markup
- `style.css` — styling for the form and layout
- `Screenshot 2024-07-03 085631.png` — example screenshot used in this README

## Usage
1. Clone or download this repository:

   git clone https://github.com/BinaryVortex/Login-Form-39.git

2. Open `index.html` in your browser (double-click the file or use "Open File" in your browser).

3. Optionally edit `style.css` and `index.html` to customize the look and behaviour.

## Customization
- Change colors, fonts and spacing in `style.css` to match your project's branding.
- Add real authentication by wiring the form to a backend or authentication service.
